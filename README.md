# Universal Calculator

Мобильное приложение для решения задач численных методов и вычислительной математики, разработанное на React Native и Expo.

## Возможности

### Решение нелинейных уравнений

* Метод простой итерации
* Метод бисекции (половинного деления)
* Метод Ньютона
* Пошаговое отображение вычислений

### Численное интегрирование

* Метод прямоугольников
* Метод трапеций
* Метод Симпсона
* Формулы Ньютона-Котеса
* Работа как с аналитическими функциями, так и с табличными данными

### Численное дифференцирование

* Первая производная
* Вторая производная
* Вычисления по набору точек

### Дифференциальные уравнения

* Метод Эйлера
* Метод Рунге-Кутты 4-го порядка
* Отображение промежуточных шагов решения

### Интерполяция

* Полином Лагранжа
* Построение графика интерполяционной функции
* Оценка погрешности интерполяции

### Системы линейных уравнений

* Решение СЛАУ методом итераций
* Поддержка ввода уравнений в текстовом виде
* Отображение процесса вычислений

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/your-username/universal-calculator.git
cd MobileCalculator
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск проекта

```bash
npx expo start
```

Или:

```bash
npm run android
```

```bash
npm run ios
```

```bash
npm run web
```

## Примеры использования

### Решение уравнения

Введите функцию:

```text
x^3 - x - 2
```

Интервал:

```text
1;2
```

Точность:

```text
0.001
```

Выберите метод и получите корень с подробными шагами вычисления.

### Вычисление интеграла

Функция:

```text
sin(x)
```

Интервал:

```text
0;3.14159
```

Количество разбиений:

```text
100
```
